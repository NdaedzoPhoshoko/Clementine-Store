# Clementine Store

Clementine Store is a full-stack e‑commerce application consisting of a React (Vite) frontend and a Node.js (Express) backend. The system provides product browsing, cart and checkout flows, order management, and administrative product controls.

Live demo: https://clementine-store.vercel.app/

Repository layout
- ecommerce-frontend/ — React + Vite user interface (client)
- ecommerce-backend/ — Node.js + Express API and server-side logic

For development and deployment instructions, see the README files in the `ecommerce-frontend` and `ecommerce-backend` folders.
